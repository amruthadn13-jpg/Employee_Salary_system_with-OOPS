# Employee_Salary_system_with-OOPS

# 💼 Day 21 – Employee Salary System (OOP in Python)

This project is a Python-based **Employee Salary System** built using Object-Oriented Programming (OOP). It demonstrates how to use classes, objects, constructors, and methods to model real-world scenarios.

---

---

## 📌 Features

* Create employee objects with name and salary
* Display employee details
* Calculate bonus based on salary
* Uses class and methods for clean structure
* Demonstrates real-world OOP usage

---

## 🧠 Concepts Used

* Classes and Objects
* Constructor (`__init__`)
* `self` keyword
* Methods inside class
* Conditional statements

---

## ⚙️ How It Works

1. A class `Employee` is created

2. Constructor initializes:

   * Employee name
   * Salary

3. Methods:

   * `display()` → shows employee details
   * `check_bonus()` → calculates bonus

4. Objects are created for employees

5. Methods are called to perform operations

---

## 💡 Bonus Logic

* Salary ≥ 50,000 → 10% bonus
* Salary < 50,000 → 5% bonus
